# Schemas

**Status:** normative artifact namespace.

This directory contains machine-readable protocol schemas and schema metadata.

## Required files for each released protocol version

- schema documents (format-specific)
- version mapping entry in `VERSION_MAP.md`
- migration notes entry in `CHANGELOG.md`

## Traceability

Schema changes must link to:
- affected spec sections (`specs/`)
- RFC (`rfcs/`)
- conformance updates (`reference-test-vectors/`, `compliance-tests/`)
